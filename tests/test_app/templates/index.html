This is the test project for django-multistep-select.

django-multistep-select aims to provide a flexible multistep selection form widget for Django's Forms API. It can help with the arbitrary filtering of choices, as well as serve as a representation of a GenericForeignKey field.

  • If you just want to use the widget, you don't really need this test project; just install the app into your own project, as per the included instructions.
  • If you would like to run the included unit tests, you would need to do this from the command line.
  • Finally, the following demo pages are included to facilitate development: filter demo; generic demo.
